PSD2

XS2A

To support the EU directive PSD2 the Berlin Group initiative described and specified the process to implement PSD2 in an open and transparent manner.

Adorsys, a development consultancy specialized in the financial and insurance sector made an XS2A blueprint and Java reference implementation of the Berlin Group specification available on GitHub. We created a fork at JavaMoney and already contributed some improvements back to the upstream repository by Adorsys via PRs. We expect to continue contributing and welcome other members of the JavaMoney community if they wish to participate.
