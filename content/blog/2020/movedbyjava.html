Moved by Java

Java 25th Birthday Week Release Train

Today Java turned 25. Oracle dedicated a Moved by Java anniversary page and kicked off the #MovedbyJava campaign on social media.
As JSR 354 Maintenance Leads we celebrated the 25th birthday of Java and the 5th anniversary of the Final Release ten days ago our own way by delivering a whole release train:

  • Money API 1.1
  • JavaMoney Parent 1.1
  • Moneta 1.4
  • TCK 1.1
  • JavaMoney Library 1.1
Check the issues resolved by the MR1 API and RI on GitHub.

While they are not parts of the actual JSR, the backports:

  • API BP 1.0.4
  • Moneta BP 1.4
also got updates.

To the next 5, 10 or 25 years of Java and JavaMoney.
