Release 1.4.1

Valkyrie Update Release

Update Release 1 of JSR 354 RI 1.4 was released today.
On this date, the 76th anniversary of the attempted "Operation Valkyrie" coup by Claus von Stauffenberg and his co-conspirators we dedicate this release to their courage, sacrifice and memory.
The release fixed a problem with the JPMS (Jigsaw), ensures compatibility with Java 14 and added a few minor improvements, for a complete list see 1.4.1 Milestone. And check the release tag 1.4.1 in the GitHub repository.

Thanks everyone who helped with this release.
