Release 1.4.4

1.4 Update 4

Update 4 of JSR 354 RI 1.4 was released today.
It is dedicated to the International Holocaust Remembrance Day.
This release fixed a critical problem with the IMF Exchange Rate Provider. Please update if you want to use both ECB and IMF exchange rates!
For a complete list see 1.4.4 Milestone. And check the release tag 1.4.4 in the GitHub repository.

Thanks everyone who helped with this release.
